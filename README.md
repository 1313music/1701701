# 1701701

李志音乐与视频站点（React + Vite）。

## 本地开发

```bash
npm install
npm run dev
```

默认开发地址：`http://localhost:8080`

## 常用命令

```bash
npm run lint
npm run test
npm run build
npm run check
```

## 项目结构

- `src/components`：页面与 UI 组件
- `src/hooks`：播放器、主题、toast 等逻辑
- `src/data`：曲库、视频、下载数据
- `src/styles`：样式文件
- `public`：静态资源（favicon、logo、sitemap 等）
